# TodoList
Todo list management system
רשימות לעשות (ToDoList): מנהל משימות פשוט

זה פרויקט לניהול משימות שרץ דרך הקונסולה. הדבר החשוב הוא שהוא שומר את כל הנתונים שלכם בקובץ **JSON** חיצוני. ככה, גם אם תסגרו את התוכנה, המשימות נשמרות ולא הולכות לאיבוד.


---

מה אפשר לעשות? (התפריט)

- להוסיף משימה חדשה: עם כותרת ותיאור, ומקבלים מספר מזהה (ID) אוטומטי.
- לעדכן משימה: שינוי פרטים או סטטוס של משימה קיימת, לפי ה-ID.
- למחוק משימה: מחיקה לגמרי.
- להציג משימות: מראה משימה ספציפית, או את כל הרשימה.
- לחפש משימה: מוצא משימות לפי טקסט מסוים.
- לסמן כסיום: דרך מהירה להעביר משימה לסטטוס 'בוצע'.
- למיין: מראה את הרשימה מסודרת לפי מה שבחרתם (למשל, לפי סטטוס).

---

איך זה מחולק? (מבנה הקוד)


- **Task:** מחזיק את הנתונים של משימה אחת (ID, כותרת, סטטוס, וכו').
- **TaskRepository:** זה המחסן. אחראי על קריאה וכתיבה לקובץ JSON ועל הפעולות הבסיסיות של הוספה/מחיקה.
- **TaskService:** המוח. עושה את הדברים הקצת יותר מורכבים, כמו מיון וחיפוש.
- **Main:** התפריט שאתם רואים והכניסה לתוכנה.



איך מפעילים?

פשוט מפעילים את מחלקת **Main.java** בסביבת הפיתוח שלכם.
